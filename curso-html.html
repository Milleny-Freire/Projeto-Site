HTML é a linguagem de marcação padrão utilizada para criar e estruturar páginas na web. Por meio de elementos chamados tags, o HTML define como textos, imagens, vídeos, links e outros conteúdos devem ser apresentados em um navegador. Ele não é uma linguagem de programação, mas sim uma linguagem que organiza e dá significado ao conteúdo das páginas, formando a base de toda a internet moderna.

Com as CSS , criamos as configurações visuais que um site vai ter.
